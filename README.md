# 🚀 LokalLens AI - Futuristic AI for Indonesian UMKM

LokalLens AI adalah platform cerdas berbasis Artificial Intelligence yang dirancang khusus untuk memberdayakan UMKM Indonesia. Kami menghadirkan teknologi visualisasi produk kelas dunia, strategi pemasaran terotomatisasi, dan analisis tren pasar real-time dalam satu ekosistem yang futuristik dan mudah digunakan.

## ✨ Fitur Utama

### 1. 🔍 Pemantau Tren Pasar (Intelijen AI)
- Analisis tren pasar real-time khusus wilayah Samarinda.
- Visualisasi data interaktif menggunakan grafik performa produk.
- Rekomendasi kata kunci (hashtags) viral untuk meningkatkan jangkauan.
- Mata-mata harga kompetitor untuk penentuan posisi pasar yang tepat.

### 2. 🪄 Visualizer Studio
- Ubah foto produk sederhana menjadi foto komersial profesional secara instan.
- Teknologi AI Image Enhancement untuk memperbaiki pencahayaan dan latar belakang.
- Berbagai gaya visual (Cafe aesthetic, Minimal Studio, dll) yang disesuaikan dengan brand lokal.

### 3. ✍️ Marketing Hub (Smart Copywriter)
- Generator caption media sosial yang persuasif dan menggunakan nuansa bahasa lokal.
- Kalkulator Harga Psikologis (Psychological Pricing) untuk memaksimalkan perceived value pelanggan.
- Integrasi langsung untuk berbagi konten ke Instagram.

### 4. 📅 Smart Scheduler
- Kalender pemasaran terintegrasi untuk mengatur jadwal promosi secara rapi.
- Analisis waktu posting terbaik (Best Time to Post) berbasis data AI.
- Manajemen inventaris jadwal promosi untuk berbagai platform (Instagram, TikTok, WhatsApp).

### 5. 🖼️ Galeri Foto AI
- Ruang penyimpanan cloud untuk hasil visualisasi produk.
- Unduh hasil karya AI berkualitas tinggi secara instan.

## 🛠️ Tech Stack

- **Frontend**: Next.js 15+ (App Router), TypeScript, Tailwind CSS, Framer Motion.
- **UI Components**: Shadcn UI, Lucide Icons, Hugeicons.
- **AI Engine**: Google Gemini API & KoboILLM Integration.
- **Database**: PostgreSQL (Neon/Local).
- **Styling**: Modern Glassmorphism & Cyber-Clean Aesthetic.

## ⚙️ Cara Menjalankan Project

1.  **Clone Repository**
    ```bash
    git clone [repository-url]
    cd lokallens-ai
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Setup Environment Variables**
    Buat file `.env` di root direktori dan masukkan:
    ```env
    DATABASE_URL=your_postgresql_url
    GEMINI_API_KEY=your_gemini_api_key
    KOBOLLM_API_KEY=your_koboillm_api_key
    AUTH_SECRET=your_auth_secret
    ```

4.  **Jalankan Migrasi Database**
    ```bash
    npm run migrate
    ```

5.  **Run Development Server**
    ```bash
    npm run dev
    ```
    Buka [http://localhost:3000](http://localhost:3000) di browser Anda.
